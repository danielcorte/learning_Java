7. Linguagem de programação, léxica, sintática
  Conjunto de regras léxicas (ortografia) e sintáticas (gramáticas) para se escrever programas

    léxica exemplo: cachorro = certo | caxorro = errado || main = certo | maim = errado

    sintática exemplo: o cachorro está com fome = certo | a cachorro está com fome = errado || "x = 2 + y" = certo | "x = + 2 y" = errado


8. IDE - Ambiente Integrado de Desenvolvimento
  Conjunto de softwares utilizados para a criação de um programa

    Exemplos:
      C/C++ = Code Blocks
      Python = Pycharm, VScode
      Java = Eclipse, NetBeans, IntelliJ

  Funcionalidades da IDE:
    Edição de código fonte (endentação, autocompletar, destaque de palavras, etc..)
    Depuração e testes
    Construção do produto final (build)
    Sugestão de modelos (templates)
    Auxiliar em varias tarefas do seu projeto
